# useful

Пересечение массивов строк.

```go
func GetIntersection(a []string, b []string, mode byte) []string 
```



mode:
```sh
1 - те что есть только в левом массиве
2 - те что только в правом
3 - пересечение
4 - это те что в пересечение не попали
```
